## Python为什么这么受欢迎？

Python是一种高级编程语言，以其简洁的语法和强大的功能而闻名。它在数据科学、人工智能、Web开发等领域都有广泛应用。

### Python的特点

- 📖 **语法简洁**：代码可读性强
- 🧩 **丰富的库**：NumPy、Pandas、TensorFlow等
- 🌐 **跨平台**：支持Windows、Mac、Linux
- 👥 **活跃的社区**：丰富的资源和教程

## 基础语法示例

```python
# 打印Hello World
print("Hello, World! 🌸")

# 变量和数据类型
name = "李娅"
age = 21
is_student = True

# 列表操作
fruits = ["苹果", "香蕉", "草莓"]
fruits.append("葡萄")

# 条件语句
if age >= 18:
    print(f"{name}是成年人")
else:
    print(f"{name}是未成年人")

# 循环
for fruit in fruits:
    print(f"我喜欢吃{fruit}")
```
